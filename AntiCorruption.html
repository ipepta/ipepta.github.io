1.通用结构

1.1通用数据表结构

Content-Type : text/plain

{
  "tableSchema":[
                  {
                    "name": "importName",
                    "chineseName": "导入名称",
                    "enumName": "",
                    "isDate": false
                  },
                  ...
                ],
  "tableData":[
           {
             "importName": "18年9月3日 10点5分 管理员 本地上传导入",
             "createdDate": "2018年09月03日 10点05分",
             ....
           },
           ...       
         ],
  "totalCount": 200
}
名称 类型 描述
tableSchema list 表头列表
name string 字段名
chineseName string 显示名
enumName string 枚举名. 配合5.2接口翻译枚举. 为空时不适用
isDate bool 是否为日期字段
tableData list 数据列表, 其中每个对象为一行数据
totalCount int 查询总数(分页用)

1.2通用参数和返回值

表格类查询均适用以下参数

名称 类型 描述 是否必填
start int 起始数据index(注意不是页码) 否, 默认0
limit int 页长度 否, 默认20

所有接口返回结果通过http状态码来标识

200代表成功，400代表请求参数错误，500代表服务器错误，401表示登录超时

类型为enum的参数/返回值请使用5.2接口进行翻译.
