1.桥梁结构

1.1桥梁结构查询

request

GET /Bridge/BridgeInfo?bridgeId=5ef490d2-1e64-4122-a44f-d02eb7d3941c

名称 类型 描述 是否必填
bridgeId string 桥梁ID 是

response

成功 200

Content-Type : text/plain

{
  "bridgeId": "5ef490d2-1e64-4122-a44f-d02eb7d3941c",
  "bridgeName": "南京夹江大桥",
  "stationList": [
                   {
                     "stationId": "21d05c86-89ca-44d6-8102-70051d735f3e",
                     "stationName": "1#空气制备站"
                   }
                   ...
                 ] 
}
名称 类型 描述
bridgeId string 桥梁ID
bridgeName string 桥梁名称
stationList List 制备站列表
stationId string 制备站ID
stationName string 制备站名称

失败 400/401/500

{
  "info": "失败描述"

3.数据上传

3.1Socket数据上传

request

POST /SubmitData/FromSocket

Content-Type : text/plain

{
  "socketJson": "{
                  "socketId": "0b89aff1-68ed-4b5e-82e1-e9a5aea1aa97",
                  "messageId": "2f4983f6-4c7c-4238-864f-3f613b448134",
                  "version": 3389,
                  "command": 0,
                  "dataCount": 3,
                  "dataList":[
                               {
                                  "dataItemIndex": 0,
                                  "id": 7500,
                                  "value": 15.50,
                                  "timeStamp": "2019/02/20 22:03:59",
                                  "exception": null
                               },
                               ...
                             ],
                  "dataHex": "33464339344530312D4334363.....",
                  "exception": null.
                  "createdDate": "2019-03-03T22:50:10.4906372+08:00",
                  "listenerSignature": "75bd62014d393719b29fca6c22b15dad"   
                }"
}

出于socket数据特殊性, 本接口所有字段全部必填, exception字段除外. exception仅当数据产生错误时传值.

名称 类型 描述
socketJson string socket数据json串
socketId string socket请求ID(服务端)
messageId string 消息ID(客户端)
version int 消息版本
command int 命令码
dataCount int 数据数量
dataList List 数据列表
dataItemIndex int 数据序号
id int 数据字段ID
value double 数据字段值
timeStamp string 数据时间
exception object socket消息错误或数据项错误
dataHex string 原始数据
createdDate datetime socket消息时间
listenerSignature string socket监听器签名

response

成功 200

失败 400/401/500

{
  "info": "失败描述"
}

6.表格

6.1通用表格结构

response

成功 200

Content-Type : text/plain

{
  "tableSchema":[
                  {
                    "name": "importName",
                    "chineseName": "导入名称",
                    "enumName": "",
                    "isDate": false,
                    "mergeAdjacentSameValueRows": true,
                  },
                  ...
                ],
  "tableData":[
           {
             "importName": "18年9月3日 10点5分 管理员 本地上传导入",
             "createdDate": "2018年09月03日 10点05分",
             ....
           },
           ...       
         ],
  "totalCount": 200
}
名称 类型 描述
tableSchema list 表头列表
name string 字段名
chineseName string 显示名
enumName string 枚举名. 配合5.2接口翻译枚举. 为空时不适用
isDate bool 是否为日期字段
mergeAdjacentSameValueRows bool 当紧邻的数据行本字段相等时,是否将这些行的本列合并在一格显示
tableData list 数据列表, 其中每个对象为一行数据
totalCount int 查询总数(分页用)

失败 400/401/500

{
  "info": "失败描述"
}
