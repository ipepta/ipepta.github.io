1.intacct saleforce sync

1.1.run sync

request

POST /IntacctSaleforceSync/RunSync

CONTENT-TYPE: application/json

{
  "saleforceUserId": "sheila@kerrconsulting.com",
  "saleforcePassword": "password12345",
  "intacctUserId": "jackyxu",
  "intacctPassword": "password54321",
  "intacctCompanyId": "KerrConsulting-szhang",
  "startLastModifyDate": "2019-03-01 00:00:00",
  "endLastModifyDate": "2019-03-31 23:59:59"
}
name type description mandatory
saleforceUserId string saleforce user id yes
saleforcePassword string saleforce password yes
intacctUserId string intacct user id yes
intacctPassword string intacct password yes
intacctCompanyId string intacct company id yes
startLastModifyDate datetime start of last-modify-date query period in saleforce yes
endLastModifyDate datetime end of last-modify-date query period in saleforce no, current time by default

date time format: yyyy-MM-dd HH:mm:ss

date time example: 2019-07-15 22:30:15
