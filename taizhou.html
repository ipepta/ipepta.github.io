1.推送数据接口

1.1登录

request

POST /Auth/Login

Content-Type : application/json

{
    "userId":"taizhouUser",
    "password": "df6!36D169sp8l"
}
名称 类型 描述 是否必填
userId string 用户名 是
password string 密码 是

response

成功 200

登陆凭据:

Cookie: ASP.NET_SessionId

失败 400/401/500

{
  "info": "失败描述"
}

1.2按设备组查询实时数据

request

GET /DeviceData/Station?stationId=2a0c0516-bb00-48b5-a17d-5eb9d8970648&isSubscribe=true

名称 类型 描述 是否必须
stationId string 设备组ID 是
isSubscribe bool 订阅查询模式, 仅返回温湿度数据 否

response

成功 200

Content-Type : text/plain

{
    "7c495c52-6e00-484a-b720-7b43bed03258":{
        "dataId":"22172440-cd99-4546-b2d5-db0670043729",
        "deviceId":"7c495c52-6e00-484a-b720-7b43bed03258",
        "value":39.28,
        "createdDate":"2023-08-11T02:15:59.3731211+08:00",
        "dataDate":"2023-08-11T02:15:53",
        "dataType":"PERCENTAGE",
        "errorLevel":"Normal"
    },
    "5315cb8a-bac3-4519-9c3d-aa9643d7742f":{
        "dataId":"1db60826-2bb8-4cef-9895-92f7c4f5a643",
        "deviceId":"5315cb8a-bac3-4519-9c3d-aa9643d7742f",
        "value":49.02,
        "createdDate":"2023-08-11T02:15:59.3731211+08:00",
        "dataDate":"2023-08-11T02:15:53",
        "dataType":"VALUE",
        "errorLevel":"Normal"
    },
  ...
}
名称 类型 描述
字典key string 设备ID
字典value object 设备数据
名称 类型 描述
dataId string 数据ID, 数据中断时为null
deviceId string 设备ID
value double 数据值, 数据中断时为null
createdDate datetime string 数据提交至系统时间
dataDate datetime string 数据采集时间
errorLevel enum 报警级别

报警级别: NORMAL, WARNING, ERROR: 正常, 警告(数据中断), 错误(数据超限)

失败 400/500

{
  "info": "失败描述"
}
