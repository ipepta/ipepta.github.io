1.应收账款(AR)接口

1.1创建应收账款

request

POST /AccountReceivable/GenerateInvoice

content-type: application/json

{
  "batchDescription": "SAGE API BATCH 0",
  "batchDate": "2019-05-12 22:30:00",
  "customerNumber": "1100",
  "invoiceDescription": "SAGE API INVOICE 0",
  "documentDate": "2019-05-12 22:31:00",
  "postingDate": "2019-06-12 10:00:00",
  "documentNumber": "DOC_00153347280",
  "optional_courseName": "MATH",
  "optional_startDate": "2019-08-01 10:00:00",
  "optional_studentId": "080233590",
  "optional_studentName": "Zhang Lin",
  "overrideBatchInfoWhenExists": false,
  "revenueAndAmountList": [
                            {
                              "revenueAccount": "1000",
                              "extendedAmount": "500.00"
                            }
                          ]

}
名称 类型 描述 是否必填
batchDescription string batch名称 是
batchDate datetime batch时间 是
customerNumber string 客户编号 是
invoiceDescription string invoice名称 是
documentDate datetime document时间 是
postingDate datetime post时间 是
documentNumber string 文档编号(不可重复) 是
optional_courseName string 可选字段 课程名称 是
optional_startDate datetime 可选字段 起始时间 是
optional_studentId string 可选字段 学生id 是
optional_studentName string 可选字段 学生姓名 是
overrideBatchInfoWhenExists bool 当使用已有batch时, 是否覆盖batch名称 batch时间 是
revenueAndAmountList list 付款金额与账户列表 是
revenueAccount string 付款账户 是
extendedAmount string 付款金额 是

时间类型格式: yyyy-MM-dd HH:mm:ss

response

成功 200

Content-Type : text/plain

{
  "batchNumber": "79",
  "entryNumber": "2",
}
名称 类型 描述
batchNumber string invoice所在batch序号
entryNumber string invoice所在entry序号

失败 400/401/500

{
  "info": "失败描述"
}

1.3查询batch

request

GET /AccountReceivable/Batch?batchNumber=95

名称 类型 描述 是否必填
batchNumber string batch编号 是

response

成功 200

Content-Type : text/plain

{
    "batchNumber": "95",
    "batchDate": "5/11/2019 12:00:00 AM",
    "batchDescription": "sage api test batch 11",
    "entryCount": "0",
    "batchTotal": "0.000",
    "batchStatus": "DELETED"
}
名称 类型 描述
batchNumber string batch编号
batchDate string batch时间
batchDescription string batch名称
entryCount string entry数量
batchTotal string batch总额
batchStatus string batch状态

batch状态: OPEN, POSTED, DELETED, POST_IN_PROGRESS, READY_TO_POST

失败 400/401/500

{
  "info": "失败描述"
}

1.4查询应收账款/核销账款

request

GET /AccountReceivable/Document?documentNumber=API_DOCUMENT_FA535DF4-&customerNumber=1100

名称 类型 描述 是否必填
documentNumber string 文档编号 是
customerNumber string 客户编号 否

当客户编号在sage中作为联合主键使用时, 必填客户编号, 否则可能会导致错误查询

response

成功 200

Content-Type : text/plain

{
    "documentNumber": "API_DOCUMENT_FA535DF4-",
    "batchNumber": "97",
    "customerNumber": "1100",
    "invoiceDescription": "sage api test invoice 11",
    "documentDate": "5/19/2019 12:00:00 AM",
    "postingDate": "6/18/2019 12:00:00 AM",
    "documentType": "CREDIT_NOTE",
    "applyToDocumentNumber": "DOC_00153347280",
    "optional_courseName": "sage api test course 11",
    "optional_startDate": "5/19/2019 12:00:00 AM",
    "optional_studentId": "sage api test student 11",
    "optional_studentName": "sage api test student aaa",
    "entryIndex": "9",
    "revenueAndAmountList": [{
        "revenueAccount": "1000",
        "extendedAmount": "200"
    }, {
        "revenueAccount": "1000",
        "extendedAmount": "100"
    }],
    "batchStatus": "POSTED"
}
名称 类型 描述
documentNumber string 文档编号
batchNumber string batch编号
customerNumber string 客户编号
invoiceDescription string 应收账款/核销账款名称
documentDate string 文档日期
postingDate string posting日期
documentType string 文档类型
applyToDocumentNumber string 核销目标文档编号(仅当查询目标为核销账款时)
optional_courseName string 可选字段 课程名称
optional_startDate datetime 可选字段 起始时间
optional_studentId string 可选字段 学生id
optional_studentName string 可选字段 学生姓名
entryIndex string 文档所在entry编号
revenueAndAmountList list 付款金额与账户列表
revenueAccount string 付款账户
extendedAmount string 付款金额
batchStatus string batch状态

batch状态: OPEN, POSTED, READY_TO_POST

文档状态: INVOICE, DEBIT_NOTE, CREDIT_NOTE

失败 400/401/500

{
  "info": "失败描述"
}
